## 🔑 核心概念（2分鐘）

### 什麼是提示工程？

> **簡單定義**：提示工程就是學會「如何更好地與AI對話」的技術。

想像一下，如果您要請朋友幫忙，您會：
- ❌ 說：「幫我做點什麼」
- ✅ 說：「請幫我分析這份報告的主要重點，並整理成3個要點」

提示工程也是如此 - 越清楚、越具體的指令，AI的回答就越有用。

### 🌟 最重要的一個技巧

**「請一步一步思考」** - 這是提示工程中最基礎也最有效的技巧！

#### 為什麼有效？
- 🧠 讓AI模擬人類的思考過程
- 🔍 將複雜問題分解成簡單步驟
- ✅ 大幅提高回答的準確性

> 💡 **白話解釋**  
> **為什麼「請一步一步思考」這麼神奇？** 就像您做菜時會先準備材料、再炒菜、最後調味一樣，把複雜的事情分成小步驟會更不容易出錯。AI也是一樣！如果直接問它複雜問題，它可能會「想太快」而出錯。但如果要求它慢慢想、一步步來，就像讓它把思考過程「說出來」一樣，答案會準確很多。

#### 使用方法
只需要在問題後面加上：
- 「請一步一步思考」
- 「Let's think step by step」
- 「請逐步分析」

## 💻 實作練習（2分鐘）

### 練習1：基礎數學問題

讓我們用一個簡單例子來體驗差異：

<table>
<tr>
<th width="50%">❌ 基礎提示</th>
<th width="50%">✅ 改進提示</th>
</tr>
<tr>
<td valign="top">

```
計算 25 × 16 是多少？
```

**典型回答**：
"25 × 16 = 400"

</td>
<td valign="top">

```
請一步一步計算 25 × 16：
```

**改進回答**：
"讓我一步一步計算：
1. 25 × 16 = 25 × (10 + 6)
2. = 25 × 10 + 25 × 6  
3. = 250 + 150
4. = 400

答案是 400。"

</td>
</tr>
</table>

### 練習2：實際問題解決

<table>
<tr>
<th width="50%">❌ 模糊提示</th>
<th width="50%">✅ 清晰提示</th>
</tr>
<tr>
<td valign="top">

```
我想減肥，給我一些建議。
```

**問題**：
- 太籠統
- 缺乏個人化
- 沒有具體目標

</td>
<td valign="top">

```
我是30歲上班族，想在3個月內健康減重5公斤。
請一步一步為我制定計劃：
1. 飲食調整建議
2. 運動計劃
3. 生活習慣改善
4. 進度追蹤方法

請考慮我平日工作繁忙的情況。
```

**優點**：
- 背景明確
- 目標具體
- 要求結構化回答
- 考慮實際限制

### 🧪 立即測試

現在輪到您了！請在您常用的AI工具（ChatGPT、Claude等）中測試以下提示：

```
我想學習Python程式設計，完全沒有程式基礎。
請一步一步為我規劃一個月的學習計劃：
1. 第一週應該學什麼？
2. 第二週的重點是什麼？ 
3. 第三週如何進階？
4. 第四週如何整合練習？

請為每週推薦具體的學習資源和練習項目。
```

## ✅ 效果驗證（1分鐘）

### 📊 自我檢核清單

請確認您的AI回答是否包含以下特點：

- [ ] **結構清晰**：有明確的步驟或章節
- [ ] **內容具體**：提供實際可行的建議
- [ ] **邏輯連貫**：前後步驟有關聯性
- [ ] **個人化**：考慮了您提到的具體情況

### 🎯 成功指標

如果您的測試結果：
- ✅ **得到了結構化的4週計劃** - 恭喜！您已經掌握基礎技巧
- ✅ **每週都有具體的學習內容** - 這就是好提示的效果
- ✅ **包含了推薦的學習資源** - 您的指令足夠清晰具體

如果沒有達到預期效果，嘗試：
1. 讓指令更加具體
2. 明確說明輸出格式要求
3. 添加相關的背景資訊
